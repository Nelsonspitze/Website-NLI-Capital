Logo rules
Always use the official SVG files: nli-capital-logo.svg (for light backgrounds) and nli-capital-logo-wit.svg (for dark backgrounds). Do not recreate the logo in custom fonts. Minimum size: 120px wide for digital / 30mm for print. Always maintain clear space equal to the height of the "nli" letterform on all sides.

Guidelines

Usage Notes

Practical rules for applying this design system consistently across all NLI Capital documents.

Do
  • Use teal as the primary brand color throughout
  • Keep coral strictly for logo "nli" and alert accents
  • Use UPPERCASE + letter-spacing for labels and slide headings
  • Use Manrope 800 ExtraBold for all display and slide titles
  • Maintain whitespace — never fill every pixel
  • Use the teal separator rule (2px line) under content headings
  • Align to 8px grid for all spacing decisions
  • Include page number and logo in every slide footer
Don't
  • Don't use coral as a background color
  • Don't mix font weights randomly — stick to the scale
  • Don't use more than 3 type sizes per slide
  • Don't recreate the logo — always use the SVG file
  • Don't use borders heavier than 1–2px
  • Don't use gradients outside of dark slide backgrounds
  • Don't use red/green for positive/negative — use teal/coral
  • Don't compress or stretch the logo
